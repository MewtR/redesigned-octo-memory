Simple Pendulum Simulator

Instructions:

  • You can set the different pendulum parameters by filling out the forms below each one and hitting the "Set" button.
  • Set "Length" and "Initial Angle" for simple harmonic motion.
  • Setting all parameters will result in a damped harmonic motion simulation.
  • Once you are satisfied with the configuration, you can click the "Export" button to save the parameters as a JSON file.
  • "Simulate" starts the simulation and "Stop" pauses it.
  • If you wish to reset the simulation, just click the "Set" button again with the same or different values.
